# Login Authentication System

## Objective

A simple Login Authentication System built using Node.js, Express, bcrypt, express-session, HTML, CSS, and JavaScript.

## Features

- User Registration
- Password Validation
- Duplicate Username/Email Check
- Secure Password Hashing using bcrypt
- Login Authentication
- Session Management
- Protected Dashboard
- Logout
- Form Validation

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- bcrypt
- express-session

## Installation

```bash
npm install
```

## Run Project

```bash
npm run dev
```

Open:

```
http://localhost:5000
```
